Dynamical Model for Transcription

This page describes examples of how to use the Dynamical Model for Variational Inference of Protein Concentration and Regulatory Intensities detailed in . The code is available for download here.

Release Information

Current release is 0.1.

Release 0.1 is the first release associated with the paper, containing scripts for recreating the results given.

The toolboxes required to run the code are listed below.

Toolbox Version
NDLUTIL
 0.15

Finally you will also need the NETLAB toolbox in your path.

Results in the Paper

The main scripts are demTuVar and demSpellmanVar which run the code on the metabolic and cell cycle data sets. Also of interest may be the script demFakeVar which runs the code on artificial data. The function chipVarTransFact computes the posterior, gene specific, transcription factor activities for a given transcription factor.
